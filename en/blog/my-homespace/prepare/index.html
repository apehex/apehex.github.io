The USB Drives

Prepare the USB drives:
- one for the live install
- one for the non-free firmware
- one for the bootloader (optional)

The Live Install

# dd bs=4M status=progress oflag=sync if=~/downloads/debian.iso of=/dev/sdd

The Non-Free Firmware

# mount /dev/sde1 /mnt/usb
# unzip ~/downloads/firmware.zip -d /mnt/usb/

Format a boot drive (optional)

# fdisk -c -u /dev/sdf

Backup Your Files

There's no may-be, data loss is guaranted!
Hopefully you're feeling the urge to backup this time?

Encrypt Your Files

$ gpgtar --verbose --encrypt --recipient moodule@protonmail.com \
        --output /tmp/backup/documents.gpg ~/documents

Save To A Drive

# rsync -avh --progress --update /tmp/backup/ /mnt/usb/backup/

_

Repeat for every directory you care about.

You should do this regularly. Instead of keeping old folders / data around, pack it up and breath.
